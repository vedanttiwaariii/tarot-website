# Product Overview - Krushnalaya Spiritual Services

## Project Purpose
Krushnalaya is a complete full-stack spiritual services platform that enables practitioners to offer Tarot Reading, Reiki Healing, and Water Divination services with integrated online booking and payment processing. The platform bridges traditional spiritual practices with modern technology to provide a seamless experience for both practitioners and seekers.

## Value Proposition
- **For Practitioners**: Professional online presence with automated booking management, secure payment processing, and administrative tools to manage appointments and client communications
- **For Spiritual Seekers**: Easy-to-use booking system with multiple payment options, transparent pricing, and immediate confirmation for spiritual guidance sessions
- **For Business**: Complete payment integration with Razorpay supporting cards, UPI, net banking, and wallets with secure transaction handling

## Key Features and Capabilities

### Client-Facing Features
- **Service Booking System**: Interactive booking forms with date/time slot selection for three service types (Tarot Reading, Reiki Healing, Water Divination)
- **Payment Integration**: Complete Razorpay payment gateway with multiple payment methods (cards, UPI, net banking, wallets)
- **Responsive Design**: Mobile-first approach with mystical UI featuring deep purple, midnight blue, and gold accents
- **Contact System**: Direct communication channel with form validation and spam protection
- **Access Code System**: Special booking codes to skip payment for promotional or VIP clients

### Administrative Features
- **Admin Dashboard**: Comprehensive booking and message management interface
- **Booking Management**: View, filter, and manage all appointments with status tracking (pending, confirmed, completed, cancelled)
- **Payment Tracking**: Monitor payment status and transaction details
- **Contact Management**: Review and respond to client inquiries
- **Bulk Operations**: Cancel all bookings or delete cancelled bookings for administrative control

### Technical Features
- **Security Hardening**: Rate limiting, XSS protection, NoSQL injection prevention, CORS configuration, Helmet security headers
- **Payment Security**: Razorpay signature verification, webhook handling, comprehensive error management
- **Form Validation**: React Hook Form with autofill support and Express Validator on backend
- **Data Validation**: Comprehensive input sanitization and validation on both frontend and backend
- **Error Handling**: User-friendly error messages with detailed logging for debugging

## Target Users and Use Cases

### Primary Users
1. **Spiritual Practitioners**: Tarot readers, Reiki healers, water diviners who need professional online booking
2. **Spiritual Seekers**: Individuals seeking guidance through tarot, healing through Reiki, or insights through water divination
3. **Site Administrators**: Business owners managing appointments, payments, and client communications

### Key Use Cases
- **Book a Session**: Client selects service, chooses date/time, provides details, completes payment, receives confirmation
- **Manage Bookings**: Admin views upcoming appointments, updates status, handles cancellations
- **Process Payments**: Secure payment collection with automatic verification and status updates
- **Client Communication**: Contact form submissions reviewed and managed through admin dashboard
- **Special Access**: VIP or promotional bookings using access codes to bypass payment

## Business Model
- Service-based revenue with upfront payment collection
- Flexible pricing structure managed through admin panel
- Support for promotional access codes for marketing campaigns
- Test and production modes for safe deployment and testing
